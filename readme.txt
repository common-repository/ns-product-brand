Add product brand to your WooCommerce products!

== Description ==

This plugin allow you to add Brand to your product in WooCommerce.
After activate this plugin you have to go in Products->Brands in your back end menu.
Here you can add your Brand! Than go into a product and associate it with brands created before.
In single product page now you can see brands.
Brand page will be created automatically.


PREMIUM FEATURES of Product Brand:


- Add shortcode that shows all brands image in a slider
- Add widget with a slider that shows all brands image link to own archives
- Add widget that shows all brands link to own archives
- Add an external link


You can **add shortcode that shows all brands image in a slider**.
Add image **Widget** and text **Widget**.
Add possibility to add **external link** to brand.

== Installation ==

1. Upload the entire 'ns-custom-product-brand' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on Products->Brands


== Frequently Asked Questions ==

= Code is required to use this plugin? =

No code require to use this plugin.

= WooCommerce is required for use this plugin? =

Yes this plugin will only work with WooCommerce plugin.

= Page Brand don't work or show me 404 =

Go to Settings->Permalinks and click on "Save Changes" to refresh permalink.
